## 修复 httpx 版本不兼容导致 ChatOpenAI 初始化失败

- 错误信息：`Client.__init__() got an unexpected keyword argument 'proxies'`
- 根因：`httpx 0.28.1` 移除了 `proxies` 参数，而 `openai==1.16.2` 内部依赖旧的 `proxies` 参数
- 修复：在 `requirements.txt` 新增 `httpx>=0.25.0,<0.28.0` 约束，降级 httpx 到 0.27.2
- 验证：proxies 报错已消除
