# Delta: source-code-reader

## MODIFIED Requirements

### Requirement: 递归目录扫描

系统 SHALL 使用自定义递归函数扫描用户指定的目标项目根目录，在每层目录中优先按文件名字典序处理源码文件，然后按目录名字典序递归进入子目录。新增断点恢复能力：当进度记录显示某个目录已全部处理完毕时，跳过该目录。

#### Scenario: 跳过已完成的目录

- **WHEN** 递归扫描过程中进入某个子目录
- **AND** 进度记录显示该目录下所有文件和子目录均已处理完毕
- **THEN** 跳过该目录，不再递归进入
- **AND** 打印跳过提示
